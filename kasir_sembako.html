Program Kasir Toko Sembako

Program sederhana untuk menghitung total belanja dengan fitur diskon

Perhitungan Otomatis

Menghitung total belanja dengan cepat dan akurat

Sistem Diskon

Diskon otomatis untuk pembelian di atas nominal tertentu

Struk Belanja

Menampilkan rincian belanja dengan format yang rapi

Penjelasan Elemen Program

Variabel & Array

  • string namaBarang[100] - Array untuk menyimpan nama-nama barang
  • int jumlah[100] - Array untuk menyimpan jumlah setiap barang
  • double harga[100] - Array untuk menyimpan harga per item
  • double total - Variabel untuk menyimpan total belanja
  • double diskon - Variabel untuk menyimpan nilai diskon

Perhitungan

  • subtotal = harga[i] * jumlah[i] - Menghitung subtotal per item
  • total += subtotal - Menambahkan subtotal ke total belanja
  • diskon = 0.1 * total - Menghitung diskon 10%
  • diskon = 0.05 * total - Menghitung diskon 5%
  • total - diskon - Menghitung total akhir setelah diskon

Struktur Kontrol

  • for loop - Untuk input dan menampilkan data barang
  • if-else - Untuk menentukan besaran diskon berdasarkan total belanja
  • Kondisi diskon:
    • ≥ Rp. 500.000 → Diskon 10%
    • ≥ Rp. 200.000 → Diskon 5%

Format Output

  • setw() - Mengatur lebar kolom output
  • left - Rata kiri untuk teks
  • fixed - Format angka desimal
  • endl - Pindah baris baru

Kode Program

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    // Deklarasi variabel
    string namaBarang[100];
    int jumlah[100];
    double harga[100], total = 0;
    int n;
    double diskon = 0;
    
    cout << "===== PROGRAM KASIR TOKO SEMBAKO =====" << endl;
    cout << "Masukkan jumlah jenis barang: ";
    cin >> n;
    
    // Input data barang
    for(int i = 0; i < n; i++) {
        cout << "\nBarang ke-" << i+1 << endl;
        cout << "Nama Barang  : ";
        cin.ignore();
        getline(cin, namaBarang[i]);
        cout << "Harga        : Rp. ";
        cin >> harga[i];
        cout << "Jumlah       : ";
        cin >> jumlah[i];
    }
    
    // Hitung total
    cout << "\n========= STRUK BELANJA =========" << endl;
    cout << "No  Barang\t\tHarga\tJumlah\tSubtotal" << endl;
    cout << "----------------------------------------" << endl;
    
    for(int i = 0; i < n; i++) {
        double subtotal = harga[i] * jumlah[i];
        total += subtotal;
        cout << i+1 << "   " << left << setw(15) << namaBarang[i] 
             << "Rp." << setw(8) << harga[i]
             << setw(8) << jumlah[i]
             << "Rp." << subtotal << endl;
    }
    
    // Hitung diskon
    if(total >= 500000) {
        diskon = 0.1 * total; // Diskon 10%
    } else if(total >= 200000) {
        diskon = 0.05 * total; // Diskon 5%
    }
    
    // Tampilkan total dan diskon
    cout << "----------------------------------------" << endl;
    cout << "Total Belanja     : Rp." << total << endl;
    cout << "Diskon            : Rp." << diskon << endl;
    cout << "Total Pembayaran  : Rp." << total - diskon << endl;
    cout << "========= TERIMA KASIH =========" << endl;
    
    return 0;
}

Fitur Program

  • Input multiple barang dengan nama, harga, dan jumlah
  • Perhitungan subtotal untuk setiap barang
  • Sistem diskon otomatis:
    • Diskon 10% untuk pembelian ≥ Rp. 500.000
    • Diskon 5% untuk pembelian ≥ Rp. 200.000
  • Tampilan struk yang rapi dengan format tabel
