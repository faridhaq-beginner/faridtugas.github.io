Program Data Diri & Tipe Data

Pengenalan berbagai tipe data dalam pemrograman melalui contoh data diri

Penjelasan Tipe Data

String

Digunakan untuk menyimpan teks seperti nama siswa

Contoh: "Muh. Farid Haq Al B."

Integer (int)

Digunakan untuk menyimpan bilangan bulat seperti umur

Contoh: 16

Float

Digunakan untuk bilangan desimal dengan presisi rendah seperti berat badan

Contoh: 65.5

Double

Digunakan untuk bilangan desimal dengan presisi tinggi seperti nilai

Contoh: 95.75

Character (char)

Digunakan untuk menyimpan karakter tunggal seperti golongan darah

Contoh: 'A'

Boolean (bool)

Digunakan untuk nilai true/false seperti status siswa aktif

Contoh: true

Kode Program

#include <iostream>
#include <string>
using namespace std;

int main() {
    // Deklarasi variabel dengan berbagai tipe data
    string nama = "Muh. Farid Haq Al B.";
    int umur = 16;
    float beratBadan = 65.5;
    double nilai = 95.75;
    char golonganDarah = 'A';
    bool statusAktif = true;

    // Menampilkan data
    cout << "====== DATA DIRI SISWA ======" << endl;
    cout << "Nama          : " << nama << endl;
    cout << "Umur          : " << umur << " tahun" << endl;
    cout << "Berat Badan   : " << beratBadan << " kg" << endl;
    cout << "Nilai         : " << nilai << endl;
    cout << "Golongan Darah: " << golonganDarah << endl;
    cout << "Status Aktif  : " << (statusAktif ? "Ya" : "Tidak") << endl;
    cout << "==========================" << endl;

    return 0;
}
